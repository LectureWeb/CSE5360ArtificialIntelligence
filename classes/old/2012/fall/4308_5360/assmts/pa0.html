Programming Assignment 0

This assignment is to test if students have all the tools necessary for their other assignments
  • Log into omega.uta.edu
  • Create a simple program to take user input and display the result on your preferred programming language (that is supported by omega) and see if it compiles
  • Compress the code file into a '.zip' archive along with a readme.txt file containing a description of the code and instructions to compile and run it (including how to provide input and output format)
  • Upload it on UTA blackboard (elearn.uta.edu)
  • Contact instructor or TA if you have any issues.
